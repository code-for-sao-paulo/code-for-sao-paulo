Somos uma brigada do Code for All que acredita no uso da tecnologia para gerar benefícios para os cidadãos. Queremos que os cidadãos participem ativamente na geração de soluções para desafios públicos.

Através do slack nós trocamos ideias de como criar uma cidade melhor. A gente também se reune de tempos em tempos através do meetup, mas se pra você for dificil comparecer, curtir nossa página do facebook já ajuda bastante a divulgar nosso trablho.
